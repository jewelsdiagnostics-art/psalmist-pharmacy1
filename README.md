# Psalmist Pharmacy Management System

## Quick Deployment Instructions

### Files Ready for Netlify Deployment:
- `index.html` - Main application
- `psalmist-script.js` - JavaScript functionality

### Deploy to Netlify:
1. Create a new GitHub repository
2. Upload these 2 files to the repository root
3. Connect repository to Netlify
4. Deploy - no build configuration needed

### Important Note:
Make sure not to include a `netlify.toml` file in your repository, as it is not needed for this project.

### Access Your System:
- Default Admin: `admin` / `admin123`
- Default Pharmacist: `pharmacist` / `pharm123`

### Features:
- Complete pharmacy inventory management
- User authentication and role-based access
- Sales processing with receipts
- CSV/Excel import/export
- Petroleum commission pricing (admin only)
- Real-time dashboard and analytics

This folder contains everything needed for immediate deployment to Netlify or any static hosting platform.
