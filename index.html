Google Sheets Sync - No API Key Required!

Step 1: Make Your Google Sheet Public

  1. Open your Google Sheet
  2. Click "Share" (top right)
  3. Change "Restricted" to "Anyone with the link"
  4. Set role to "Viewer"
  5. Click "Done"

Your sheet: https://docs.google.com/spreadsheets/d/1hg_N2it72QP11WTiVW6tJ5DUE1TqRqNrhUcUydJMyPs/edit

Step 2: Set Up Sheet Structure

Create three tabs with these exact headers:

Tab: "Users"

ID | Username | Password | Role | Name | Enabled | LastUpdated

Tab: "Drugs"

ID | Name | Strength | Cost | GeneralPrice | PetroleumPrice | Stock | MinBal | Location | WhenTaken | Expiry | LastUpdated

Tab: "Activity"

Timestamp | Message | User

Step 3: Add Initial Data

Add this data to get started:

Users Tab Data:

1|admin|admin123|admin|Administrator|TRUE|<current date>
2|pharmacist|pharm123|pharmacist|Pharmacist|TRUE|<current date>
3|accountant|acct123|accountant|Accountant|TRUE|<current date>
4|doctor|doc123|doctor|Doctor|TRUE|<current date>

Step 4: Test the Sync

  1. Open your pharmacy system in Chrome
  2. Login as admin (admin/admin123)
  3. Add a new drug
  4. Open pharmacy system in Firefox
  5. The drug should appear automatically!

How It Works:

  • Public Google Sheet acts as database
  • No API key needed - uses public CSV export
  • Real-time sync across all browsers
  • All users see identical data
  • Automatic backups in Google Sheets

No API key required - just make your sheet public!
